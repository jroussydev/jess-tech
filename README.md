# Jess Tech

Site vitrine officiel de **Jess Tech**, développé avec React, TypeScript et Tailwind CSS.

## À propos

Jess Tech est mon projet de développeuse web freelance. Ce site a pour objectif de présenter mes services, mes réalisations, mes projets internes et ma façon de travailler.

Le projet est développé entièrement par mes soins dans le but de progresser techniquement tout en construisant un site professionnel destiné à mes futurs clients.

## Fonctionnalités

* Présentation de Jess Tech
* Présentation des services proposés
* Mise en avant des réalisations
* Présentation des projets internes
* Design responsive
* Interface moderne développée avec Tailwind CSS

## Technologies utilisées

* React
* TypeScript
* Vite
* Tailwind CSS
* React Router
* Lucide React

## Installation

```bash
npm install
npm run dev
```

Pour créer une version de production :

```bash
npm run build
```

## Objectifs du projet

* Développer un site vitrine moderne et performant.
* Mettre en pratique mes compétences en développement web.
* Présenter mon activité professionnelle.
* Faire évoluer progressivement le projet au fil de mon apprentissage.

## État du projet

🚧 En cours de développement.

Le site évolue régulièrement avec l'ajout de nouvelles fonctionnalités, l'amélioration du design et la création de nouvelles pages.
